 ngsweb

Source Code


ngsweb is a webserver designed around modularity and to be easily expandable.
ngsweb allows for easy configuration of the website's structure, you can always be sure the users are seeing exactly what you want them to see.

Want an example? Well well, you're already seeing it in action! This entire site (Including NGS's land and the hudder wiki) is run with ngsweb.

If you want to take a peak at how ngsweb works in action feel free to visit this website's source code
