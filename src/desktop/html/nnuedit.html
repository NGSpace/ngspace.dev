NNUEdit is a code editor I made with the goal of one day reviving this website/project and help me rebuild it from scratch as well as add some games and any future games/projects to this list (I use Zed editor now).

It is completely open source and available and deeply tested in both windows and linux. It should work on MacOS too but it was never tested there.

It is very well optimized to minimize cpu, gpu and memory usage, even when working with (extremely) large files (on my pc it was able to read files up to 2GB big without lagging a bit!)

It supports more than 40 programming langauges!

It is very compact and easy to use!

Execute shell scripts and html files with F5!
